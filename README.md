# React Notes Demo

A simple React application created as a learning and demonstration project.

## Overview
This project is a basic notes application where users can:
- Add notes
- View notes
- Delete notes

The data is stored using browser localStorage.

## Purpose
This project was built to practice and demonstrate:
- React fundamentals
- useState and useEffect
- Handling form input
- Rendering lists
- Basic CRUD logic on the frontend

## Tech Stack
- React JS
- JavaScript
- HTML
- CSS

## Deployment
This project is deployed using Vercel.
